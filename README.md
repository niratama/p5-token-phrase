# NAME

Text::TokenPhrase - Simple module that generates unique phrases

# SYNOPSIS

    use Text::TokenPhrase;

# DESCRIPTION

Text::TokenPhrase is Perl port of [TokenPhrase](https://github.com/genericsteele/token\_phrase).
